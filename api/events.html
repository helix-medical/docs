Middlewares

!!! info "Middlewares" The middlewares are used to check if the request body is valid. Return a 406 NOT_ACCEPTABLE error if the request body is invalid. Else, the request is passed to the next middleware.

Create

Check if the request body is valid.

{
  "start": "YYYY-MM-DD HH:mm",
  "end": "YYYY-MM-DD HH:mm",
  "calendar": "[0-9a-fA-F]{8}",
  "appointment": "[0-9a-fA-F]{8}", // Not required
  "title": "string",
}

Update Date

Check if the request body is valid.

{
  "start": "YYYY-MM-DD HH:mm",
  "end": "YYYY-MM-DD HH:mm",
}

Update Calendar

Check if the request body is valid.

{
  "calendar": "[0-9a-fA-F]{8}",
}

Endpoints

Read All

Return all events.

[
  {
    "id": "[0-9a-fA-F]{8}",
    "start": "YYYY-MM-DD HH:mm",
    "end": "YYYY-MM-DD HH:mm",
    "calendar": "[0-9a-fA-F]{8}",
    "appointment": "[0-9a-fA-F]{8} || ''",
    "title": "string",
  },
  ...
]

Read One

Return one event.

{
  "id": "[0-9a-fA-F]{8}",
  "start": "YYYY-MM-DD HH:mm",
  "end": "YYYY-MM-DD HH:mm",
  "calendar": "[0-9a-fA-F]{8}",
  "appointment": "[0-9a-fA-F]{8} || ''",
  "title": "string",
}

Create Endpoint

Create an event.

{
  "start": "YYYY-MM-DD HH:mm",
  "end": "YYYY-MM-DD HH:mm",
  "calendar": "[0-9a-fA-F]{8}",
  "appointment": "[0-9a-fA-F]{8}", // Not required
  "title": "string",
}

{
  "id": "[0-9a-fA-F]{8}",
  "message": "Event ${id} created"
}

Update Date Endpoint

Update the date of an event.

{
  "start": "YYYY-MM-DD HH:mm",
  "end": "YYYY-MM-DD HH:mm",
}

{
  "id": "[0-9a-fA-F]{8}",
  "message": "Event ${id} updated"
}

Update Calendar Endpoint

Update the calendar of an event.

{
  "calendar": "[0-9a-fA-F]{8}",
}

{
  "id": "[0-9a-fA-F]{8}",
  "message": "Event ${id} updated"
}

Delete

Delete an event.

{
  "id": "[0-9a-fA-F]{8}",
  "message": "Event ${id} deleted"
}
